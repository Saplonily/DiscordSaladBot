## How to build
Use .net core 3.1 to build it.<br>
Easy like **`dotnet build`**<br>
Remember to create a new file named `config.json`<br>
And its content:<br>
```json
{
    "config": {
        "token": "-Your app Token here-",
        "resource_header": "/salad-resource"
    }
}
```
